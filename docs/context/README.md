# 项目级上下文索引

这个目录用于给人类与 agent 提供项目级上下文，而不是功能细节文档。

建议阅读顺序：
1. `PROJECT_CONTEXT.md`：项目是什么、现在到哪一步、核心对象与关键路径
2. `OPEN_QUESTIONS.md`：当前未定事项、风险、待确认点
3. `WORKFLOW_KNOWLEDGE_BASE.md`：workflow 家族如何组织、如何拆 stage、如何给 agent 使用
4. `WORKING_AGREEMENTS.md`：协作约定、修改边界、落盘习惯

维护原则：
- 优先记录“当前真实状态”，不要写理想化描述
- 一旦主流程、目录结构、关键命令变化，同步更新这里
- 这不是 changelog；这里只保留对后续工作最重要的上下文
