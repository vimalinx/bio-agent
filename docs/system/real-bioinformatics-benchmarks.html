Real Bioinformatics Benchmarks

这一页定义 `bio-agent` 当前用来“做题”的真实 benchmark 面：不是跑 marketing demo，而是拿官方 challenge、consortium benchmark、或标准化 conformance task 来检验 workflow routing、plan 结构和 analysis-flow 覆盖。

当前 benchmark 层先评估 3 件事：route 对不对、plan 的 expected outputs / confirmation gates 对不对、analysis flow 的 delivery bundle 是否覆盖到 benchmark 任务要求。它还不直接评估真实数值指标本身。

How To Run

当前最小使用方式有两种：

benchmark-report benchmark-evaluate benchmark-run benchmark-suite 
python3 scripts/bio_skill_system.py benchmark-report

python3 scripts/bio_skill_system.py benchmark-evaluate \
  --benchmark-id giab-precisionfda-small-variant \
  --task-id giab-wes-bwa-gatk-hardfilter

python3 scripts/bio_skill_system.py benchmark-run \
  --benchmark-id giab-precisionfda-small-variant \
  --task-id giab-wes-bwa-gatk-hardfilter \
  --evidence-file examples/benchmarks/giab-wes-bwa-gatk-hardfilter.evidence.json

python3 scripts/bio_skill_system.py benchmark-run \
  --benchmark-id giab-precisionfda-small-variant \
  --task-id giab-wes-bwa-gatk-hardfilter \
  --evidence-file examples/benchmarks/giab-wes-bwa-gatk-hardfilter.happy.evidence.json

python3 scripts/bio_skill_system.py benchmark-suite \
  --mode contract

python3 scripts/bio_skill_system.py benchmark-suite \
  --benchmark-id giab-precisionfda-small-variant \
  --mode evidence \
  --evidence-root examples/benchmarks

python3 scripts/bio_skill_system.py benchmark-evaluate \
  --task-file examples/benchmarks/encode-atac-bowtie2-macs3.task.json

benchmark-evaluate 评估控制面契约；benchmark-run 在此基础上再结合 session 或 evidence 文件里的真实交付物、truth artifacts、metric keys 生成一份 scorecard；benchmark-suite 会把多道题批量跑完，输出一份聚合 score report。对 GIAB 这类题，当前 scorecard 还会解析 hap.py summary.csv 这类常见 benchmark artifact，提取 precision / recall / F1 数值，检查范围并做 F1 一致性校验。
