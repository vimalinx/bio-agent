Real Bioinformatics Workflow Map

这一页只收录可追溯到官方或成熟社区工作流的流程家族，用来给 bio-agent 的 request normalization、workflow matching、candidate plan generation 提供 grounded 参考。

当前优先落地的是 4 类高价值流程：bulk RNA-seq、scRNA-seq、ATAC-seq、germline short variant discovery。它们的输入边界、阶段结构和输出合同都比较稳定，最适合转成 agent workflow。对应的真实分析信息流已单独整理在 real-analysis-information-flows.html。

为什么先做这些官方/社区文档成熟，阶段化强，输入输出边界清楚，适合自然语言到 plan 的半自动调度。
这页不做什么不把营销文案当流程，不把 broken repo 说明当能力，也不默认所有候选工具在本仓库已落地可执行。

Agent 映射

这些真实流程可以统一压进你当前 bio-agent 的 stage skeleton：

s1 input-and-reference-check
s2 heavy compute / alignment / matrix build
s3 intermediate artifact generation
s4 statistics / peak / genotype / clustering
s5 report-review

高算力、覆盖写入、外部下载、参考资源不明确的阶段，默认都应该挂到 execution-guard 的确认点。
