Real Analysis Information Flows

这一页把 grounded workflow family 的真实分析信息流显式展开，不只说有哪些 stage，而是说每个 stage 消耗什么、产生什么、在哪里形成证据、最终交付包应该长什么样。

这层的目标是让 bio-agent 同时理解控制流和分析流。用户批准的不只是 stage 顺序，也是在批准一条 artifact handoff 和 evidence accumulation 路径。

架构位置workflow registry 负责 family routing；analysis flow registry 负责真实数据移动和交付语义。
当前覆盖先覆盖 4 条已 grounded 且已 routeable 的 family：bulk RNA-seq、scRNA-seq、ATAC-seq、germline short variant discovery。

Routeable Flows

这 4 条 flow 已同时具备 grounded workflow 定义、strategy profile、example plan/request，以及显式 analysis-flow 定义。
